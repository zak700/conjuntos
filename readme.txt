Desenvolvido pelo matemático George Cantor, conjuntos é um conceito primitivo da matemática que hoje em dia é a principal base da álgebra e de conceitos de grande importância e de outros conceitos importantíssimos na matemática, como funções e inequações.
A notação que usamos para conjuntos é normalmente uma letra maiúscula do alfabeto latino (por exemplo A, B, C, ETC...)
para descrever os elementos englobamos todos entre chaves e os separamos com vírgulas. Também é possível resumir com reticências caso o conjunto tenha um padrão claro, por exemplo:
A = { a, b, c, d, e, f, ..., y, z }
Neste caso A é o conjunto das letras do alfabeto Brasileiro.
