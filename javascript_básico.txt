JavaScript (js) é uma linguagem de progração usada principalmente para adicionar interatividade a páginas web, mas também pode ser usada em multiplas outras coisas como, por exemplo servidores, sendo utilizadas em diversos sites e jogos durante os anos.

porém antes de começar a aprender sobre JavaScript é necessário entender o que é o console.

O console refere-se a uma interface de linha de comando, onde o desenvolvedor pode interagir com o código, executar comandos e visualizar mensagens de depuração. E ao escrever códigos é necessário o testar de maneira prática e utilizar o console para fazer justamente isso, testar o código atual.

No JavaScript existem multiplas funções, dentre elas tem o console.log() 
O principal objetivo desta função é exibir informações durante o desenvolvimento, ajudando os desenvolvedores a inspecionar variáveis, rastrear caminhos de execução e identificar problemas em seu código.
para utilizar coloque a informação diretamente dentro dos parenteses, por exemplo:

console.log("amarelo") // saida: amarelo

o terminal(saida) irá retornar a palavra amarelo, significando que o código funcionou corretamente

VARIAVEIS

No JavaScript as váriaveis são espaços de memória no computador que o programa reserva para ser executado eventualmente.
Existem, no total, Três tipos de váriaveis que podemos utilizar.

var;
let; e
const.

é fundamental saber a diferença entre elas para o uso correto de tais.

VAR

O JavaScript possui várias características, dentre elas tem uma chamada 'hoisting' isso significa que quando temos uma váriavel do tipo var, ela é levantada para o topo do seu escopo (refere-se ao contexto em que as variáveis e funções são acessíveis dentro do código), por exemplo:

var cor = "roxo"
console.log(cor) // saida: roxo
