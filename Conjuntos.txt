Desenvolvido pelo matemático George Cantor, conjuntos é um conceito primitivo da matemática que hoje em dia é a principal base da álgebra e de conceitos de grande importância e de outros conceitos importantíssimos na matemática, como funções e inequações.
A notação que usamos para conjuntos é normalmente uma letra maiúscula do alfabeto latino (por exemplo A, B, C, ETC...)
para descrever os elementos englobamos todos entre chaves e os separamos com vírgulas. Também é possível resumir com reticências caso o conjunto tenha um padrão claro, por exemplo:
A = { a, b, c, d, e, f, ..., y, z }
Neste caso A é o conjunto das letras do alfabeto Brasileiro. 

Existem operações com os conjuntos, alguns exemplos de operações basicas com conjuntos são: União (U), Intercecção (N), Diferença (-).

A União sendo representada pelo simbolo "U", é responsavel por unir os elementos de dois ou mais conjuntos, por exemplo:
T = {1, 3, 5} L = {2, 4}, T u L = {1, 2, 3, 4, 5}                                                                                     lembrando que quando numeros se repetem eles não entram na União, por exemplo:
H = {1, 2} e F = {1, 3} H u F = {1, 2, 3}

A Intercecção sendo representada pelo simbolo "(N)", é responsável por destacar apenas os elementos que se repetem nos conjuntos que estão nessas operações, por exemplo:
T = {1, 2, 3, 4}  L = {1, 2, 3, 5} T n L = {1, 2, 3}

A Diferença sendo representada pelo simbolo "-", é responsável por destacar somente os elementos que não tem em comum nos conjuntos que estão nessas operações, por exemplo:
T = {1, 2, 3, 4}  L = {1, 2, 4, 5, 6}  T - L = {3} 
Deixando claro que a ordem dos fatores altera o final, os elementos não em comum que aparecem no conjunto final devem ser apenas os do primeiro conjunto da operação. então se fosse L - T o resultado seria {3, 5, 6}
